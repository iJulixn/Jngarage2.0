# JN Garage Detail - App PRO

Proyecto listo para subir a Vercel o GitHub.

## Cómo correrlo local
```bash
npm install
npm run dev
```

## Cómo subirlo a Vercel
- Subí esta carpeta a GitHub
- Conectá el repo con Vercel
- Framework: Vite
- Build command: npm run build
- Output directory: dist
